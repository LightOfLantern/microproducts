Products

Focused tools instead of bloated platforms

Microproducts ships compact software for specific professional tasks. Our products may be distributed as web tools, Chrome extensions, Figma plugins, App Store apps, or Google Play apps. Each commercial product is designed to be easy to understand, easy to buy, and easy to cancel when recurring billing applies.

AutoRedactor

AutoRedactor is a Figma plugin that uses AI to improve UX and interface text directly inside selected frames. It helps product designers and teams make copy shorter, clearer, and more consistent without leaving the design file.

Figma plugin AI text editing Digital access

Screenshotter

Screenshotter is a Chrome extension for automated capture on configured target websites. It can save visible-page PNG screenshots or full-page PDF exports after page load, organize files in the Downloads folder, track capture statistics, use a configurable delay, and skip excluded URLs.

Chrome extension PNG and PDF capture URL exclusions

Shelf

Shelf is a browser extension for keeping prompts in one place. It helps users save, organize, and reuse prompt snippets without searching through notes, chats, or scattered documents.

Browser extension Prompt storage Quick reuse
